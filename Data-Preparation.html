Data Preparation

- All job codes with the code “317” had been removed as that code does not exist.

- All job codes with the code “0” and “9999” had been changed to “000”

- All the job codes with the code “312(full time)” and “411(part time)” have been edited to “411” and “517” respectively

- Every character had been turned into lower case letters

- All the characters had been reverted back to their common stem
